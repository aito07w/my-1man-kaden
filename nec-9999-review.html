01. PROCESSOR

CPU：演算装置の「遺産」

Intel Core i5-4000M Series (Haswell)

処理性能の目安 実用レベル
50% (日常OK)

※ネット閲覧や動画視聴ならストレスなく動作します。

【主な仕様】

コア / スレッド数 2コア / 4スレッド
ベース周波数 2.5GHz 〜 2.7GHz
L3キャッシュ 3MB Intel Smart Cache

搭載されているのは2013年発表、第4世代Coreアーキテクチャ「Haswell」。 22nmプロセス世代のモバイル向けCPUで、当時はビジネスノートの主力。 今となっては歴史展示レベルですが、完全な骨董品というわけでもありません。

末尾が「U」ではなく「M」である点がこの個体の特徴。 TDP 37Wと現代の省電力CPUより消費電力は高めですが、 その分クロックの粘りはある。 ただしマルチコア全盛時代の今、2コア構成は明確に世代差を感じます。

現実として、最新スマホSoCに純粋性能で劣る場面もある。 それでも物理ファン搭載、交換可能パーツ構造、 そして修理可能という「機械としての誇り」は残っている。 爆速は期待するな。 だが文書作成、ブラウザ、軽作業ならまだ働く。 これは未来志向のCPUではない。 過去の遺産を再雇用するという選択だ。
02. MAIN MEMORY

メモリ：作業領域の「限界点」

DDR3L-1600 (PC3L-12800)

処理速度の目安（規格性能） 実用レベル
50% (日常OK)

※速度よりも「8GB以上の容量確保」が快適さの鍵となります。

【選択可能なメモリ構成】

4GB（標準構成） 最低生存圏 / 即メモリ圧死
8GB（＋4000円） 人権ライン / 現実的選択
16GB（＋10000円） 過保護構成 / 長寿命志向

メモリ規格は低電圧版のDDR3L-1600。 2026年基準では完全に旧世代。帯域もクロックもDDR5とは別世界。 ただし、このクラスのCPU性能を考えれば理論値を語っても意味は薄い。 問題は「何枚タブを開く気か」という現実です。 4GBはWindows 11がかろうじて息をする容量。 8GBでようやく複数作業が成立。 16GBまで積めば、この筐体におけるメモリ不足の不安はほぼ消滅します。 速度ではなく、余裕の確保がテーマです。

1万円家電ハンター
ブラウザは裏で静かにメモリを食う捕食者。 Office、Zoom、PDF、そして謎の常駐ソフト。 4GBは「使える」ではなく「動く」だけ。 8GBは妥協点。 16GBはこの筐体における精神安定剤。 ただし本体性能との釣り合いを考えると、 16GBは安心感重視の投資であって爆速化ではない。 つまり容量選択はロマンではなく、 自分の作業密度と不安耐性を測る現実テストです。
03. STORAGE UNIT

ストレージ：延命の「特効薬」

2.5インチ SATA SSD

データ読み書きの快適さ 爆速（HDD比）
80% (快適)

※OSの起動速度に関しては、最新PCに勝るとも劣らない体感速度です。

【選択可能な容量オプション】

128GB（標準構成） クラウド依存症向け / すぐ満腹
256GB（＋3000円） 一番人気 / 現実を理解した人類
512GB（＋8000円） 写真・動画・謎データ保管庫
1TB (1024GB)（＋18000円） ロマン枠 / 消さない民の終着駅

このPCが「まだ人権を保っている」最大の理由は、旧世代のHDDを潔く葬り去り、 2.5インチ SATA SSDへ換装している点にあります。 回転する円盤に祈りながら起動を待つ原始的儀式は終了。SSD化によりOS起動・アプリ起動・ アップデート処理すべてが現代水準に接近します。 要するに「展示用の遺物」を「日常労働可能な道具」に矯正する最低条件です。 例えるなら、走行5万キロオーバーの軽トラに少し前のスポーツカーのエンジンを移植した状態。 最新M.2 NVMe SSDは確かに速いですが、このクラスの本体にそこまで投資するのは 高級レーシングタイヤを三輪車に履かせるような資本配分ミスです。

現実的な運用を考えるとOffice、ブラウザ地獄、Zoom録画、謎PDFの蓄積で ストレージは確実に増殖します。256GBは「普通の人間が普通に使う」最低ライン。 128GBは軽量Linux信者か、YouTube専用端末として余生を送る覚悟がある人向け。 512GBは消すという概念を理解しない層の保険。1TBは安心感はあるが、 本体性能とのバランス的にやや過剰投資。つまり容量選択は夢ではなく、 自分の怠惰レベルを正確に測る心理テストです。

バッテリー：持ち運べない「重り」

【電力供給の実態】

公称駆動時間 測定不能（期待値ゼロ）
実効容量（推定） 10%以下（完全放電の危機）
新品交換オプション 本機種は「対象外」

中古PC市場における「バッテリー」とは、駆動装置ではなく「停電時のバックアップ用UPS」と解釈するのが大人というものです。 2013年製のセルが現代まで現役でいられるはずもなく、中身はスカスカの骨董品。 販売ページには「新品バッテリー交換可能（一部機種）」という甘い言葉が並んでいますが、 この9,999円という極限価格を実現するために、真っ先に切り捨てられたのがその「救済措置」です。

つまり、この個体における「新品交換」の文字は、手の届かない蜃気楼。 届いた瞬間に「バッテリー駆動できない！」と騒ぐのはお門違いというものでしょう。 最初からACアダプターという名の「生命維持装置」に繋がれた、動かせないノートPC。 それがこの価格の代償です。

現実を見よう。これは「持ち運べるPC」ではない。 「取っ手がついた、場所を取らないデスクトップ」だ。 バッテリーが死んでいるということは、 余計な熱源や発火リスクが低いというポジティブな（無理のある）解釈もできる。 オプションの新品交換を期待した奴は、 商品ページの「注釈」という名の迷宮に迷い込んだだけ。 このPCはコンセントに縛られる運命なんだ。 自由を求めるなら、もう5万円積んでから出直してこい。

2. 店舗評価：Harukisuは信頼できるか

中古PCで一番怖いのは、性能でも価格でもありません。
「届いた瞬間に終わる個体」です。
そこでHarukisuのショップレビュー（評価4.36）と、公式の保証規定をポジティブフィルターなしで徹底的に精査しました。

  • 発送の早さ： 「注文の翌日に届いた」という報告が多数。ここはガチです。
  • 梱包： 専用箱・緩衝材あり。少なくとも「適当に投げ込まれた」状態ではありません。
  • 外装の状態： 天板の傷は前提。キーボードが綺麗なら「当たり」扱いです。
    ※新品同様を期待すると、開封した瞬間に心が折れます。

ここが重要：鉄壁の保証・サポート体制

「安いから壊れても文句言えない？」いいえ、言えます。
Harukisuが生き残っている理由は、この「逃げないサポート体制」にあります。

① 7日以内の返金・交換対応 初期不良

届いてすぐ「電源が入らない」「画面が割れている」などの不具合があれば、到着後7日以内なら返金または交換が可能。ここが「ジャンク品」との決定的な違いです。

② 30日間の通常保証 標準保証

使っていて1ヶ月以内に調子が悪くなった場合も、修理や交換の対象になります。
※ただし、自分の過失（水をこぼした等）やバッテリーの持ち（消耗品扱い）は対象外なので注意。

絶対やるべき：レビュー特典の「裏技」

このPCを買うなら、商品到着後に必ずレビューを書いてください。
書くだけで、保証内容が「別物」に進化します。

  • 保証期間が「最大90日」に延長
    通常1ヶ月の保証が約3ヶ月に伸びます。中古PCの不具合は3ヶ月以内に大体出尽くすので、この延長はデカいです。
  • 選べるプレゼントが貰える
    無線マウスやUSBメモリなど、地味に買うと1,000円くらいする周辺機器がタダで貰えます。貰えるものは貰っておきましょう。
現在のプレゼント対象商品を確認する

結論として、Harukisuは

神対応を期待すると不満が出ますが、
