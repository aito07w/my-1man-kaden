9,999円のOffice付きPCは本当に得か？
楽天市場「Harukisu」のVersaProを冷静に解剖する

2026.01.28 更新 調査：現実を壊す1万円家電ハンター

「1万円以下で、ちゃんと動くWindows PCが欲しい」

その願い、気持ちは分かります。財布が軽いと、判断力も軽くなるものです。

楽天市場のランキングに居座り続ける「Harukisu（ハルキス）」のNEC VersaPro。 レビュー数は700件超、評価は4.2以上。数字だけ見れば優等生です。

ただし、数字は人を裏切りませんが、人は数字を誤解します。

本記事では、購入ボタンを押す前に知っておくべき 「なぜこの価格なのか」「どこを妥協しているのか」「誰が泣くのか」 を、遠慮なく全部書きます。

1. 詳細スペック分析：9,999円の中身

販売ページに書いてあるスペックを、そのまま信じるほど世の中は優しくありません。 ここでは「安い理由」をごまかさず、IT的に冷水をぶっかけながら分解します。

主要パーツ 現実を直視した解説
CPU
Brain of PC（ただし高齢）
Intel Core i5（第4世代 / Mシリーズ）

2026年基準では、完全に「ベテラン枠」のCPUです。 設計は10年以上前。最新技術？ありません。

ただし末尾の「M」は、よくある格安ノートに載っている 省電力Celeronとは違い、一瞬だけ本気を出せるタイプ。 Excelやブラウザ程度なら、 「思ったより普通に動くじゃん」と錯覚させてくれます。

期待しすぎなければ、今でも事務作業要員としては しぶとく生き残る、そんなCPUです。

メモリ
Work Space（狭い）
4GB〜16GB（DDR3）

このPC最大の地雷原です。 メモリ4GBは、例えるなら 机の上にノートPCを置いたら、もうコーヒーを置く場所がない 状態。

Windowsを起動した時点で余裕はほぼ消滅。 Chromeでタブを20枚開いた瞬間、 「さっきまで元気だったのに？」となります。

将来を少しでも考えるなら16GB一択。 ただし、それでも油断すると重くなるので、 後述する軽量化設定は義務です。

ストレージ
Storage Speed（唯一の良心）
新品 SSD 128GB

ここだけは素直に褒めます。 中古PC最大の敵であるHDDを捨て、 新品SSDに換装しているのは英断。

起動が速い、アプリがすぐ開く。 「あれ？意外と快適じゃん」と感じたら、 それは全部SSDのおかげです。 CPUを褒めてはいけません。

付属ソフト
Standard Software（本体より高い）
MS Office 2019 H&B（永続版）

正直に言います。 この商品の本体はPCではなくOfficeです。

今どきOffice単体でも1〜2万円。 つまりこれは 「Officeを買ったら、なぜか中古PCが付いてきたセット」 です。

サブスクが嫌いな人にとっては、 ここだけで元を取った気分になれるでしょう。

液晶 / 筐体
Display & Body（時代遅れ）
15.6型 / 1366×768 / 約2.4kg

解像度は今やスマホ以下。 重さは2Lペットボトル級。 持ち歩くなら覚悟が要ります。

ただし、元は法人向け。 キーボードは打ちやすく、 雑に扱っても壊れにくい。

見た目は古いが、 道具としては今でも現役。 そんな立ち位置です。

2. 店舗評価：Harukisuは信頼できるか

中古PCで一番怖いのは、性能でも価格でもありません。 「届いた瞬間に終わる個体」です。 そこでHarukisuのショップレビュー（評価4.63）を、 ポジティブフィルターを外して確認しました。

  • 発送の早さ： 速いです。これは事実。 逆に言うと「速さ」しか褒められていないレビューも多い。
  • 梱包： 専用箱・緩衝材あり。 少なくとも「投げたな？」という扱いではありません。
  • 外装の状態： 天板は傷あり前提。 キーボードが綺麗なら「当たり」扱いです。 新品気分で開封すると精神を削られます。

結論として、Harukisuは 「最低限の仕事はきちんとする中古PC屋」です。 神対応を期待すると不満が出ますが、 地雷を踏む確率は低い部類。 期待値を正しく設定できる人向けの店です。

3. 性能検証：第4世代Core i5の実力

「第4世代」という数字だけで拒否反応を示す人もいますが、 このCPU、意外としぶとい。 ただし、万能だと勘違いすると痛い目を見ます。

💡 現実的にできること・できないこと

  • ⭕ Excel・Wordなどの事務作業 → 余裕。これが本業。
  • ⭕ YouTube視聴（〜1080p） → 問題なし。ただし裏で何もするな。
  • ❌ 3Dゲーム・動画編集 → 話を戻そう。
  • 🔺 Windows 11の基本動作 → 動くが、快適とは言っていない。

このPCは「何でもできる」機械ではありません。 やることを限定した瞬間に輝く道具です。 逆に欲張ると、一気に化石に見えてきます。

4. 必須知識：Taiki式軽量化術

このPCを「普通に使える」と感じるか、 「やっぱり安物だった」と嘆くかは、 ここで全てが決まります。

断言しますが、初期状態のまま使う人は向いていません。

ChromeOS Flexに逃げる選択肢もありますが、 Officeを使いたいならWindows一択。 つまり、Windowsを軽くする努力が必須です。 これはカスタマイズではなく、通過儀礼です。

5. 総評：買うべき人・やめるべき人

買うべき人（現実が見えている）

  • Officeをとにかく安く使いたい
  • 自宅据え置きで割り切って使う
  • 設定変更を「面倒」と思わない

やめるべき人（期待が過剰）

  • カフェでドヤ顔使用したい
  • バッテリーだけで一日使いたい
  • 箱から出した瞬間に爆速を求める

結論

このPCは魔法の箱ではありません。
ただし、用途を割り切れる人にとっては、 9,999円とは思えない働きをする道具です。
