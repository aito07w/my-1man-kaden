1. スペック徹底分析：9,999円の中身

販売ページに書いてあるスペックを、そのまま信じるほど世の中は優しくありません。
ここでは「安い理由」をごまかさず、IT的に冷水をぶっかけながら分解します。

01. PROCESSOR

CPU：演算装置の「遺産」

Intel Core i5-4000M Series (Haswell)

搭載されているのは、2013年に発表された第4世代Intel Coreプロセッサ、コードネーム「Haswell」です。22nmプロセスルールで製造されたこのチップは、かつては世界のビジネスの最前線を支えていました。

特筆すべきは、末尾が「U（超低電圧版）」ではなく「M（通常電圧版）」である点。現代の薄型PC用CPUよりも消費電力（TDP 37W）は高いですが、その分、古いなりに「粘り」のある処理を見せます。

  • コア/スレッド数: 2コア / 4スレッド
  • ベース周波数: 2.5GHz 〜 2.7GHz
  • L3キャッシュ: 3MB Intel Smart Cache

※皮肉な現実：最新のスマホに処理能力で負けることもありますが、彼には「物理的な排熱ファン」というプライドがあります。

02. MAIN MEMORY

メモリ：作業領域の「限界点」

Crucial Alert

DDR3L-1600 (PC3L-12800)

メモリ規格は低電圧版のDDR3L。2026年現在の主流であるDDR5と比較すると、データの転送速度は「徒歩とリニア中央新幹線」ほどの差があります。

4GB (標準構成)

Windows 11を動かす上での「最低生存圏」。ブラウザのタブを5枚開けば、そこはもう戦場です。

8GB (推奨)

「人権」の最低ライン。事務作業、YouTube視聴を並行して行うなら、ここからがスタート地点です。

03. STORAGE UNIT

ストレージ：延命の「特効薬」

2.5インチ SATA SSD

【選択可能な容量オプション】

128GB（追加料金なし） クラウド依存症向け / すぐ満腹
256GB（＋3000円） 一番人気 / 現実を理解した人類
512GB（＋8000円） 写真・動画・謎データ保管庫
1TB (1024GB)（＋18000円） ロマン枠 / 消さない民の終着駅

このPCが「まだ人権を保っている」最大の理由は、旧世代のHDDを潔く葬り去り、 2.5インチ SATA SSDへ換装している点にあります。 回転する円盤に祈りながら起動を待つ原始的儀式は終了。SSD化によりOS起動・アプリ起動・ アップデート処理すべてが現代水準に接近します。 要するに「展示用の遺物」を「日常労働可能な道具」に矯正する最低条件です。 例えるなら、走行5万キロオーバーの軽トラに少し前のスポーツカーのエンジンを移植した状態。 最新M.2 NVMe SSDは確かに速いですが、このクラスの本体にそこまで投資するのは 高級レーシングタイヤを三輪車に履かせるような資本配分ミスです。

現実的な運用を考えるとOffice、ブラウザ地獄、Zoom録画、謎PDFの蓄積で ストレージは確実に増殖します。256GBは「普通の人間が普通に使う」最低ライン。 128GBは軽量Linux信者か、YouTube専用端末として余生を送る覚悟がある人向け。 512GBは消すという概念を理解しない層の保険。1TBは安心感はあるが、 本体性能とのバランス的にやや過剰投資。つまり容量選択は夢ではなく、 自分の怠惰レベルを正確に測る心理テストです。
04. CONNECTIVITY

インターフェース：時代の「地層」

「ドングル地獄」に陥っている最新のMacBookユーザーが、羨望の眼差し（あるいは同情）を向ける充実のポート類です。

  • USB 3.0

    計5ポート搭載。マウス、プリンタ、USBメモリ、加湿器、扇風機を同時に繋げます。

  • VGA & HDMI

    新旧両対応。15ピンのVGAがあるため、昭和から稼働しているプロジェクターにも接続可能。

  • Optical

    DVDマルチドライブ搭載。懐かしのDVD鑑賞や、音楽CDの取り込みに。令和ではもはや貴重品です。

05. CHASSIS & KEYBOARD

筐体：ビジネスの「戦車」

NEC VersaProの筐体は、華奢なモバイルノートとは一線を画す「重厚長大」な設計です。 重量は約2.4kg。持ち歩くのは大変ですが、デスクに置いた際の安定感は抜群。

打鍵感へのこだわり

キーボードは、最近流行りの浅いタイプではありません。深さ2.4mmのしっかりとしたストロークがあり、 「カチャカチャ」ではなく「スコスコ」とした伝統的なメンブレンの感触を楽しめます。テンキー付きで数値入力も捗ります。

06. POWER & NETWORK

バッテリーと通信：現実との「妥協」

🔋 バッテリーについて

中古PC最大の弱点。正直に言います。ACアダプタを抜いての長時間作業は「博打」です。 バッテリーは「停電時のバックアップ」程度に考え、常に電源を確保して使用することを強く推奨します。

📶 Wi-Fi 搭載（アダプタ同梱）

内蔵されていない、あるいは内蔵されていても現代の規格には遅い。 そのため、**USB無線LANアダプタ**を同梱します。 これにより、2.4GHz/5GHzの両方で安定した通信が可能。USBポートが一つ埋まりますが、安定には代えられません。

2. 店舗評価：Harukisuは信頼できるか

中古PCで一番怖いのは、性能でも価格でもありません。
「届いた瞬間に終わる個体」です。
そこでHarukisuのショップレビュー（評価4.36）と、公式の保証規定をポジティブフィルターなしで徹底的に精査しました。

  • 発送の早さ： 「注文の翌日に届いた」という報告が多数。ここはガチです。
  • 梱包： 専用箱・緩衝材あり。少なくとも「適当に投げ込まれた」状態ではありません。
  • 外装の状態： 天板の傷は前提。キーボードが綺麗なら「当たり」扱いです。
    ※新品同様を期待すると、開封した瞬間に心が折れます。

ここが重要：鉄壁の保証・サポート体制

「安いから壊れても文句言えない？」いいえ、言えます。
Harukisuが生き残っている理由は、この「逃げないサポート体制」にあります。

① 7日以内の返金・交換対応 初期不良

届いてすぐ「電源が入らない」「画面が割れている」などの不具合があれば、到着後7日以内なら返金または交換が可能。ここが「ジャンク品」との決定的な違いです。

② 30日間の通常保証 標準保証

使っていて1ヶ月以内に調子が悪くなった場合も、修理や交換の対象になります。
※ただし、自分の過失（水をこぼした等）やバッテリーの持ち（消耗品扱い）は対象外なので注意。

絶対やるべき：レビュー特典の「裏技」

このPCを買うなら、商品到着後に必ずレビューを書いてください。
書くだけで、保証内容が「別物」に進化します。

  • 保証期間が「最大90日」に延長
    通常1ヶ月の保証が約3ヶ月に伸びます。中古PCの不具合は3ヶ月以内に大体出尽くすので、この延長はデカいです。
  • 選べるプレゼントが貰える
    無線マウスやUSBメモリなど、地味に買うと1,000円くらいする周辺機器がタダで貰えます。貰えるものは貰っておきましょう。
現在のプレゼント対象商品を確認する

結論として、Harukisuは
「最低限の仕事はきちんとする中古PC屋」です。
神対応を期待すると不満が出ますが、
地雷を踏む確率は低い部類。
期待値を正しく設定できる人向けの店です。
